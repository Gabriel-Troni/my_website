Projetos

Pesquisa de Países

SQL | PHP | Bootstrap | CSS | HTML

Projeto desenvolvido com Matheus Grego do Amaral para a disciplina de Programação Web I. O objetivo era criar um site que se conectasse a um banco de dados e, em resposta a uma requisição POST, retornasse os resultados da pesquisa em tela seguindo os filtros, ordenação e limite de resultados inseridos pelo usuário. Todos os campos de input deveriam passar por sanitização e demais validações.

Dashboard de Vendas no PowerBI

Power BI | SQL | SSMS | Excel

Dashboard em PowerBI para análise de dados e descoberta de insights sobre as vendas de uma pizzaria. O projeto iniciou com uma tabela de excel, que teve seus dados modelados e importados para um banco de dados SQL. A partir disto, foram criadas consultas para validação dos dados que foram posteriormente apresentados no dashboard. O dashboard faz uso de algumas das principais funcionalidades do PowerBI, como ETL, fórmulas DAX, KPI, paginação, gráficos responsivos e filtros.

Crack the Code

SQL | PHP | Javascript | CSS | HTML

Projeto desenvolvido com Matheus Grego, Raul Bana e Rodolfo dos Santos para a disciplina de Programação Web I. O objetivo era criar um jogo de digitação que possuísse base de dados própria para armazenar dados de cada jogador, como: login, senha, nome e pontuação em cada partida. O jogo conta também com um sistema de ligas e ranking. Neste projeto, trabalhei principalmente com a modelagem e conexão com o banco de dados, consultas SQL e com validações em back end.

Gerenciador de Ramais

SQL | PHP | Javascript | CSS | Bootstrap | HTML

Projeto pessoal desenvolvido para o gerenciamento de ramais empresariais. O projeto conta com um sistema de login, validações de segurança e permissões. Usuários comuns podem consultar funcionários, setores e ramais, enquanto administradores podem inserir, pesquisar, editar e excluir registros (CRUD). O sistema conta com banco de dados próprio para controle dos ramais e usuários, já a pesquisa feita pelo usuário utiliza Regex.
