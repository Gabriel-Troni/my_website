Sobre mim

Nasci em Ji-Paraná, no Estado de Rondônia e foi onde vivi por 20 anos. Em 2023 me mudei para Curitiba, no Paraná, para fazer faculdade. Estou no terceiro período de Análise e Desenvolvimento de Sistemas pela Universidade Federal do Paraná (UFPR). Atualmente trabalho como Aprendiz de TI na APK Logística e Transporte. Sou apaixonado por tecnologia e por aprender coisas novas. Meus hobbies são jogar no computador, assistir animes e jogar RPG.

Projetos

Pesquisa de Países

Tecnologias utilizadas: PHP, SQL, HTML, Bootstrap e CSS

Projeto desenvolvido com Matheus Grego do Amaral para a disciplina de Programação Web I. O objetivo era criar um site que se conectasse a um banco de dados e, em resposta a uma requisição POST, retornasse os resultados da pesquisa em tela seguindo os filtros, ordenação e limite de resultados inseridos pelo usuário. Todos os campos de input deveriam passar por sanitização e demais validações.

Crack the Code

Tecnologias utilizadas: PHP, SQL, Javascript, CSS e HTML

Projeto desenvolvido com Matheus Grego, Raul Bana e Rodolfo dos Santos para a disciplina de Programação Web I. O objetivo era criar um jogo de digitação que possuísse base de dados própria para armazenar dados de cada jogador, como: login, senha, nome e pontuação em cada partida. O jogo conta também com um sistema de ligas e ranking. Neste projeto, trabalhei principalmente com a modelagem do banco de dados, consultas SQL e com validações em back end.

Projetos Front End

Tecnologias utilizadas: HTML, CSS e Javascript

Projetos desenvolvidos durante cursos, bootcamps e maratonas de programação. Entre estes projetos, estão um agregador de links (imagem), clone da Netflix e um jogo da memória. O objetivo principal foi aprender conceitos de front end, como: HTML semântico, responsividade, flexbox, grid, animações e manipulação de D.O.M.
