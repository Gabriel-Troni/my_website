Projects

Country Search

SQL | PHP | Bootstrap | CSS | HTML

Project developed with Matheus Grego do Amaral for the Web Programming I course. The goal was to create a website that connects to a database and, in response to a POST request, displays search results on the screen according to the user's input for filters, sorting, and result limits. All input fields had to undergo sanitization and additional validations.

Power BI Sales Dashboard

Power BI | SQL | SSMS | Excel

PowerBI dashboard for data analysis and insight discovery regarding a pizzeria's sales. The project began with an Excel table, where the data was modeled and imported into an SQL database. From there, queries were created to validate the data, which were subsequently presented on the dashboard. The dashboard leverages some of PowerBI's key functionalities, such as ETL, DAX formulas, KPIs, pagination, responsive graphics, and filters.

Crack the Code

SQL | PHP | Javascript | CSS | HTML

Project developed with Matheus Grego, Raul Bana, and Rodolfo dos Santos for the Web Programming I course. The goal was to create a typing game with its own database to store data for each player, such as login, password, name, and score for each game. The game also features a league and ranking system. In this project, I primarily worked on database modeling, SQL queries, and backend validations.

Extension Manager

SQL | PHP | Javascript | CSS | Bootstrap | HTML

Personal project developed for the management of corporate extensions. The project features a login system, security validations, and permissions. Common users can query employees, departments, and extensions, while administrators can create, read, update, and delete records (CRUD). The system features its own database for managing extensions and users, while user searches utilize Regex.
